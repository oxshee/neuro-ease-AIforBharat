## 📖 Overview

**NeuroEase** is a browser extension designed to make online reading **simpler, calmer, and more accessible**. Modern websites often contain distractions such as advertisements, animations, pop‑ups, and dense blocks of text. These elements can make reading difficult—especially for **neurodivergent individuals**, including people with ADHD, dyslexia, or anyone who struggles with information overload.

NeuroEase transforms cluttered webpages into a **clean, focused reading environment** using AI-powered tools and intelligent page filtering.

---

## ✨ Key Features

### 🔹 Simplify Mode

Summarizes complex or lengthy text into clear, concise information so users can quickly understand the main ideas.

### 🔹 Calm Mode

Disables moving elements such as animations, autoplay videos, and dynamic ads to reduce visual distractions.

### 🔹 Remove Distractions Mode

Removes unnecessary interface components such as ads, sidebars, and recommendation panels, leaving only the core written content.

## ⚙️ Installation

Since this extension is currently a **prototype**, it must be installed using Developer Mode.

### 1. Download the Extension

Clone the repository or download it as a ZIP file:

```
git clone https://github.com/your-username/neuroease.git
```

or download the ZIP and extract it.

### 2. Open Browser Extensions

Navigate to:

```
chrome://extensions/
```

### 3. Enable Developer Mode

Toggle **Developer Mode** in the top-right corner.

### 4. Load the Extension

1. Click **Load Unpacked**
2. Select the **browser extension folder** inside this repository

The NeuroEase extension will now appear in your browser.

---

## 🚀 Usage

1. Open any article or text-heavy webpage
2. Click the **NeuroEase extension icon**
3. Activate one of the modes:

* Simplify
* Calm
* Remove Distractions

The page will instantly transform into a more readable format.
